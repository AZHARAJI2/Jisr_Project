## 🎯 ملخص الفكرة

**جسر الأزمات** تطبيق موبايل يتيح تحويل الأموال بدون إنترنت بالكامل.

المُرسِل في منطقة انقطع إنترنتها يرسل الحوالة من تطبيقه، التحويل ينتقل من هاتف لهاتف عبر WiFi Direct والبلوتوث حتى يصل لشخص عنده إنترنت، فيرتفع للسيرفر ويصل للمستلم كإشعار فوري كأي تطبيق بنكي.

| | |
|---|---|
| **المسار** | Track 1 — إرسال واستقبال الأموال |
| **الجمهور** | مناطق الأزمات + العمالة في الخليج |
| **التقنية** | WiFi Direct + Bluetooth + AI Routing |
| **التكلفة** | صفر — كل المكتبات مجانية |
| **المنصة** | Android (Flutter) |

- مليارات تُدفع رسوماً لـ Western Union كل سنة

## ✨ الميزات الرئيسية

### 1. 🤖 المسار الذكي بالذكاء الاصطناعي

كل هاتف له درجة تُحسب من 4 عوامل:

| العامل | الوزن | التفسير |
|--------|-------|---------|
| مستوى البطارية | 40% | كلما أعلى كلما أفضل |
| قوة الإشارة | 30% | كلما أقوى كلما أفضل |
| التحويلات النشطة | 20% | كلما أقل كلما أفضل |
| القرب من الهدف | 10% | كلما أقرب كلما أفضل |

التحويل يمشي دائماً عبر الهاتف الأعلى درجةً.

**ترشيد البطارية:**
```
AI لا يبحث باستمرار —
ينام 60 ثانية → يبحث 10 ثوان → ينام → ...
يوفر 80% من استهلاك البطارية
```

### 3. 🔒 الأمان الكامل

**منع Double-Spending:**
```
أبو خالد عنده 100 ريال
يحاول يرسل 100 لخالد
وفي نفس الوقت 100 لسارة

التطبيق يخصم محلياً فوراً
من أول إرسال ← يمنع الثاني
لا يتزامن مع السيرفر إلا بعد التأكيد
```

**التشفير الكامل:**
```
كل تحويل = حزمة مشفرة
الهاتف الوسيط يحملها دون أن يفتحها
مثل ساعي البريد — يوصّل بدون يقرأ
```

```

✅ قاعدة البيانات = محفظة رقمية قانونية
   = لا تحتاجون أموالاً حقيقية في Escrow
```

## 💻 الكود الأساسي

### خوارزمية المسار الذكي
```dart
class SmartRouter {
  
  double calculateNodeScore(NetworkNode node) {
    double score = 0;
    
    // البطارية — 40%
    score += (node.batteryLevel / 100) * 40;
    
    // قوة الإشارة — 30%
    score += (node.signalStrength / 100) * 30;
    
    // التحويلات النشطة — 20%
    double transferPenalty = (node.activeTransfers * 5).clamp(0, 20);
    score += (20 - transferPenalty);
    
    // القرب من الهدف — 10%
    score += node.proximityScore * 10;
    
    return score.clamp(0, 100);
  }
  
  List<NetworkNode> findBestPath(List<NetworkNode> nodes) {
    nodes.sort((a, b) =>
      calculateNodeScore(b).compareTo(calculateNodeScore(a))
    );
    return nodes;
  }
}
```

### منع Double-Spending
```dart
class LocalWallet {
  Future<bool> sendTransfer(double amount, String toUser) async {
    // تحقق محلياً من الرصيد أولاً
    double currentBalance = await hive.get('balance');
    
    if (currentBalance < amount) {
      return false; // رصيد غير كافٍ
    }
    
    // خصم فوري محلياً — يمنع الإرسال المزدوج
    await hive.put('balance', currentBalance - amount);
    
    // أرسل عبر الشبكة
    bool sent = await meshNetwork.send(TransferPacket(
      amount: amount,
      to: toUser,
      timestamp: DateTime.now(),
      signature: generateSignature(),
    ));
    
    if (!sent) {
      // فشل الإرسال — أعد الرصيد
      await hive.put('balance', currentBalance);
      return false;
    }
    
    return true;
  }
}
```
